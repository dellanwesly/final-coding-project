# final-coding-project

I created a higher or lower cards game where the player is given a card and has to predict if the next card will be a higher or lower value. the game keeps a streak of how many were correctly guessed and once the user gets a wrong prediction the game shows the value of the card and what the streak ended at 

How to Play
-The game shows a card
-Type h for higher or l for lower
-Type q to quit
-The game continues until you get one wrong

 this must be typed into the terminal: python project.py cards.txt
